# 戏境 — 用户画像卡

## 画像 1：林悦（沉浸式体验者）

### 基本信息
| 项目 | 内容 |
|------|------|
| 姓名 | 林悦 |
| 角色 | 普通用户 — 核心体验者 |
| 年龄 | 22-28岁 |
| 技术水平 | 中高级 |
| 使用频率 | 每天/每周 |
| 使用设备 | PC写大纲+手机玩故事 |

### 核心目标
1. 体验"活在小说中"的沉浸感，自己的选择能改变故事走向
2. 完成一部属于自己的完整小说，可以分享给朋友

### 主要痛点
1. 传统小说是被动阅读，无法参与剧情走向
2. 想写小说但没耐心/没能力从零写到完
3. 现有AI写作工具只有生成，没有"互动决策"环节

### 典型使用场景
> "周末晚上窝在沙发里，用手机打开戏境，继续我的修仙故事。上一章我的角色选择救了反派，这一章AI会如何发展剧情？我迫不及待想看。"

### 关键需求
| 优先级 | 需求 | 对应功能 |
|--------|------|----------|
| P0 | 流畅的章节阅读体验 | F301 章节生成 |
| P0 | 有意义的决策点 | F302 关键决策 |
| P0 | 快速创建世界观 | F201-F204 世界观工坊 |
| P1 | 完成后导出分享 | F403 导出TXT |

## 画像 2：陈哲（设定控/创作者）

### 基本信息
| 项目 | 内容 |
|------|------|
| 姓名 | 陈哲 |
| 角色 | 普通用户 — 设定爱好者 |
| 年龄 | 25-35岁 |
| 技术水平 | 高级（会自配API） |
| 使用频率 | 每周 |
| 使用设备 | PC为主 |

### 核心目标
1. 构建复杂有趣的世界观和大纲，看AI如何演绎自己的设定
2. 尝试不同AI模型的效果差异，找到最适合叙事的模型

### 主要痛点
1. 现有AI小说工具的世界观设定太浅，无法构建有深度的体系
2. 不同AI模型的叙事风格差异大，想灵活切换但现有工具不支持
3. 设定做完后AI经常跑偏，不按大纲走

### 典型使用场景
> "花一晚上在PC上搭建了一个克苏鲁+赛博朋克混合世界观，选了DeepSeek来生成，看看这个模型的恐怖描写功底。大纲我写了12章，结局要暗黑收束。"

### 关键需求
| 优先级 | 需求 | 对应功能 |
|--------|------|----------|
| P0 | 深度世界观设定 | F202 世界设定 |
| P0 | AI补全后可编辑 | F204 AI补全设定 |
| P0 | 多模型切换 | F103 API配置 + F304 模型选择 |
| P1 | 大纲约束AI不跑偏 | F201 大纲编辑 |
