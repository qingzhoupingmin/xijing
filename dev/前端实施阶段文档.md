## 一、项目搭建（脚手架）

### 1.1 创建 Vite 项目

```bash
npm create vite@latest xijing -- --template react-ts
cd xijing
```

### 1.2 安装依赖

```bash
# 核心
npm install react-router-dom zustand axios lucide-react
npm install -D tailwindcss @tailwindcss/vite
```

## 二、全局设计令牌接入

### 2.1 复制 tokens.css

将 `dev/tokens.css` 复制到 `src/tokens.css`，在 `main.tsx` 中导入：

```tsx
// src/main.tsx
import './tokens.css'
import './index.css'
```

### 2.2 配置 Tailwind

在 `src/index.css` 中添加 Tailwind 指令：

```css
@import "tailwindcss";

/* 全局字体 */
body {
  font-family: var(--font-family);
  color: var(--color-neutral-7);
  background-color: var(--color-surface-page);
}
```

## 三、公共组件库（共 19 个）

组件按依赖顺序排列，**前面的组件不依赖后面的组件，后面的组件可能依赖前面的组件**，必须按顺序实现。

### 3.1 Button 按钮

**依赖关系**：无，所有组件的基石。

**props**：
| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| variant | `'primary' \| 'secondary' \| 'text' \| 'danger' \| 'gold' \| 'gradient'` | `'primary'` | 按钮变体 |
| size | `'sm' \| 'md' \| 'lg'` | `'md'` | 尺寸 |
| disabled | `boolean` | `false` | 禁用 |
| loading | `boolean` | `false` | 加载中 |
| children | `ReactNode` | 必填 | 内容 |
| onClick | `() => void` | - | 点击回调 |
| icon | `ReactNode` | - | 前置图标 |
| fullWidth | `boolean` | `false` | 100% 宽度 |

**尺寸规格**：
| 尺寸 | height | padding | fontSize | borderRadius |
|------|--------|---------|----------|-------------|
| sm | 28px | 8px 12px | 12px | 4px |
| md | 36px | 12px 16px | 14px | 8px |
| lg | 44px | 16px 24px | 16px | 8px |

**变体色值**：
| variant | 文字色 | 背景色 | hover 背景 |
|---------|--------|--------|-----------|
| primary | #FFFFFF | #7C5CFC | #9B82FC |
| secondary | #374151 | 透明 | #F9FAFB（边框 #E5E7EB） |
| text | #374151 | 透明 | #F3F4F6 |
| danger | #FFFFFF | #EF4444 | 略深 |
| gold | #FFFFFF | #C9984A | #DDB06E |
| gradient | #FFFFFF | linear-gradient(135deg, #7C5CFC, #9B82FC) | |

### 8.2 API 封装

```ts
// src/lib/api.ts
import axios from 'axios'

const api = axios.create({
  baseURL: import.meta.env.VITE_API_BASE || '',
  timeout: 30000,
})

// 请求拦截器：注入 token
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token')
  if (token) config.headers.Authorization = `Bearer ${token}`
  return config
})

// 响应拦截器：401 跳转登录
api.interceptors.response.use(
  (res) => res,
  (err) => {
    if (err.response?.status === 401) {
      // 跳转 /login
    }
    return Promise.reject(err)
  }
)
```

### 8.4 SSE 流式输出（p004 核心）

```ts
// 在生成章节时使用 EventSource 或 fetch + ReadableStream
const generateChapter = async (storyId: string, modelId?: string) => {
  const response = await fetch(`/api/stories/${storyId}/generate`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ modelId }),
  })

  const reader = response.body!.getReader()
  const decoder = new TextDecoder()

  while (true) {
    const { done, value } = await reader.read()
    if (done) break
    const text = decoder.decode(value)
    // 逐块更新 readingCard 内容
  }
}
```
